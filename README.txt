# TP pré-Hangman

## Pour lancer l'affichage de la promo et de ses élèves :

- Tapez dans votre navigateur, dans la barre d'URL : localhost:8080/promo

## Pour lancer l'affichage du nombre de vues sur la page :

- Tapez dans votre navigateur, dans la barre d'URL : localhost:8080/change

## Pour lancer votre formulaire d'inscription :

- Tapez dans votre navigateur, dans la barre d'URL : localhost:8080/form

*Si vous ne savez pas ce qu'est un navigateur, installez Google Chrome, ou Firefox, ou bien Yahoo.

## Informations techniques

- Ce jeu utilise le langage Golang, CSS et HTML.
- Chaque fonction possède son commentaire décrivant son utilité.
